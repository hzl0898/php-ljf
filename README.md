# php-ljfl
php垃圾分类管理系统垃圾分类指南平台，前端采用layui、vue框架技术，有用户、会员、管理员三种角色。前台功能：登录注册、垃圾识别搜索评论收藏功能、在线留言、在线考试、在线客服、个人中心管理等；后台功能：垃圾分类管理、垃圾识别管理、识别记录管理、留言管理、试题试卷管理、考试管理等。需要源码和数据库的小伙伴请联系Q：2364821220
